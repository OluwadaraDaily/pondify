About Pondify

Pondify is a final project of OLOYE OLUWADARA AYOMIDE and ABIOLA ADEOLA AMOS of the department of Electrical and Information Engineering, Covenant University, Otta, Nigeria.

This project aims to make remote monitoring of fish pond possible by integrating Hardware sensors with software application.

This project employed the use of sensors, a microcontroller, a cloud platform, various web technologies (HTML, CSS, Django, SQLite) and API to make the implementation possible.

This project was supervised by PROF. FRANCIS IDACHABA.
